# Wordlist Generator

#Overview
A Python-based wordlist generator that creates custom wordlists using user-defined inputs. This tool is useful for basic cybersecurity tasks such as subdomain enumeration, testing in controlled environments, and understanding how wordlists are used in security workflows.

#Features
- Generates wordlists from user input
- Supports combinations and variations of keywords
- Simple and easy to use
- Can be used with other tools for enumeration

#Technologies Used
- Python
- String manipulation

#How It Works
The program takes input words or patterns and generates different combinations to form a wordlist. These wordlists can be used for testing or enumeration purposes in a controlled environment.

#How to Run

1. Make sure Python is installed  
2. Run the script:
    python wordlist_generator.py
